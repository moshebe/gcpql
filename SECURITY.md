# Security

## Reporting a vulnerability

Please **do not** open a public issue for security vulnerabilities.

To report a vulnerability, open a [GitHub Security Advisory](https://github.com/moshebe/gcpql/security/advisories/new) (preferred) or email the maintainers privately if you need to disclose without using GitHub.

We will acknowledge receipt and work with you on a fix and disclosure timeline.

## Scope

- **In scope:** gcpql CLI and its dependencies (as used by this repo).
- **Out of scope:** General GCP or third-party service security; please report those to the respective vendors.

## Supported versions

We release security updates for the latest major version. Upgrade to the latest release to receive patches.
